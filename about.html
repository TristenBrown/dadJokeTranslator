Hello, my name is Tristen Brown and I am currently taking CS 2830 with Professor Wergeles at Mizzou. I created this application in order to entertain people, which is especially important now with Covid. It has four main functions. The first is generating a random dad joke pulled from an api. The second is translating that dad joke into a funny made up language using a seperate api. Sadly, since i'm using this api for free, you're only allowed 5 calls to change the language per hour. The third functionality is creating your own joke and having the option to save it. The last main functionality is signing up in order to save and load jokes and translations that you thought were especially hilarious. I hope you enjoy it!
